# Lab 06 — React JS (State, Event Handling, Routing)

## How to Run Any Task

1. Open a terminal
2. cd into the task folder
3. Run:

```
npm install
npm run dev
```

4. Open http://localhost:5173 in your browser

---

## Tasks

| Folder          | Task                         | Concepts                        |
|-----------------|------------------------------|---------------------------------|
| task1-counter   | Counter App                  | useState, Increment/Decrement   |
| task2-userform  | User Form App                | useState, onChange, Submit      |
| task3-events    | Interactive Buttons          | onClick, onMouseOver, onMouseOut|
| task4-routing   | Multi-Page Website           | React Router, Link, Routes      |

---

## Requirements
- Node.js installed (https://nodejs.org)
- Run `node -v` to verify
